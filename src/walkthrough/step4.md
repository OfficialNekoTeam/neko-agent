# Agent Modes

Neko supports different modes for different tasks.

## Available Modes

### Agent Mode
Full autonomous mode with all tools available. The AI can read, write, and execute commands.

### Plan Mode
Creates a detailed plan before executing. Good for complex tasks.

### Ask Mode
Question-answering mode. The AI can only read and analyze code, not modify it.

### Edit Mode
Focused on code changes. Direct editing without running commands.

## Switching Modes

Click the mode selector in the chat input to switch between modes.
