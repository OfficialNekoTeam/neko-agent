# Configure API Provider

To use Neko AI, you need to configure an API provider.

## Supported Providers

- OpenAI (GPT-4, GPT-3.5)
- Anthropic (Claude)
- Google (Gemini)
- Ollama (Local models)
- And many more...

## Setup Steps

1. Click the settings icon in the Neko panel
2. Select your preferred provider
3. Enter your API key
4. Choose a model

## Getting API Keys

- **OpenAI**: Visit [platform.openai.com](https://platform.openai.com)
- **Anthropic**: Visit [console.anthropic.com](https://console.anthropic.com)
- **Google**: Visit [ai.google.dev](https://ai.google.dev)
